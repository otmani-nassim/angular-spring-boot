# Description
A website that helps you save a list of servers and check if they are up.

# Technologies
## Backend
- Spring Boot
- Cors
- MySQL
- Docker (MySql server)
- Java

## Frontend
- NgRX, RjRX
- Reactive programming
- Angular (15)
- Angular notifier

# How to run it
## Backend
```
cd learnngrx
mvn spring-boot:run
```
## Frontend
```
cd serverapp
ng server
```
